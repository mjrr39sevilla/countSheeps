# countSheeps
## Descripción
Este proyecto consiste en crear una función en JavaScript que cuenta el número de ovejas presentes en una lista de valores booleanos.
Los valores `true` representan una oveja presente y los valores `false` representan una oveja ausente.
## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
## Funcionalidad
La función recibe un array de valores booleanos y devuelve el número total de elementos con valor `true`.
### Ejemplo
``` javascript
const list1 = [true, true, false, true];

countSheeps(list1);
// Resultado: 3
```
